# Spacewar II

一个基于 Godot 4.6.1 的俯视角动作原型项目，用来验证 `双武器切换 + 敌群波次推进 + 局间升级选择` 是否能形成更稳定的中程可玩性。

## 当前阶段

当前仓库实现的是第一阶段最小可玩闭环，范围刻意控制在单个战斗场景内，优先验证战斗节奏是否成立：

- 主菜单
- 单个战斗场景
- 玩家移动、主武器射击、受伤、死亡
- 副武器切换与使用
- 敌人按波次刷新
- 每波结束后的 3 选 1 升级
- HUD 状态显示
- 死亡结果页

## 运行方式

& 'D:\Development\Godot\Godot_v4.6.1-stable_win64.exe' --path 'D:\workspace4Codex\spacewar II'
```

& 'D:\Development\Godot\Godot_v4.6.1-stable_win64_console.exe' --headless --path 'D:\workspace4Codex\spacewar II' -s res://scripts/SmokeTest.gd
```

## 操作说明

- `WASD` 或方向键：移动
- `Space`：主武器射击
- `Q`：切换副武器
- `E`：使用当前副武器
- `1 / 2 / 3`：在升级面板中选择升级

## 项目结构

- `scenes/`：主要场景与实体场景
- `scripts/`：战斗逻辑、全局状态、敌人、波次、升级等脚本
- `ui/`：HUD 与局间升级 UI
- `docs/`：开发文档

## 当前完整流程

1. 从主菜单进入游戏。
2. 在战斗场景内清理一波敌人。
3. 波次结束后进入升级 3 选 1。
4. 继续推进后续波次，利用主武器和副武器维持节奏。
5. 玩家死亡后进入结果页，查看生存波次、击毁数和已选升级。
